h1. Silk is a netflow collection tool which is open source.

The configuration is the directory labled netflow is for the configuration of SILK. Silk will collect netflow data from a network which is in a binary format. The configuration that is provided in the netflow directory provides the syntax for the rwcut to create a ASCII dump of the netflow data.
